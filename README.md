I had a cool idea, and so I decided to try and make it.
If you want to help, my discord is ars_astralis.
Unfortunately, this is a passion project, as I have no money.

The core mechanic is a 2D platformer, similar to Celeste, but introducing roguelike mechanics, with power-ups
you can buy with in game currency (earned via in game events/levels/etc) at a shop for powerups, upgrade
powerups, speed, and more, in order to find secret/hidden/hard to reach areas that you wouldn't be able to get
on your first playthrough.
There will be an "endless mode" you can play, where the higher you score, the more currency you get. This
will be procedurally generated.
A "Custom Mode" and "Challenge Mode" is also set to be within this project later on, allowing different challenges
for each level.
